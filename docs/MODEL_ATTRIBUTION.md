# Model and dependency attribution

The harness code is Apache-2.0. Kokoro ONNX, Kokoro voice files, OpenAI Whisper,
Montreal Forced Aligner, eSpeak NG, FFmpeg, and their model weights are separate
third-party works. They are downloaded only after opt-in and are not committed
to this repository. Consult each upstream license and model card before use,
especially for commercial narration or redistribution.
